## Technologies & Patterns

### Frontend Stack
- **React 19** - Server Components support
- **Next.js 16** - App Router, SSR, API Routes
- **TypeScript 5+** - Complete type safety
- **Tailwind CSS** - Utility-first styling

### State Management
- **Zustand** - Lightweight client state
- **React Query** - Server state, cache, sync

### Testing
- **Playwright** - E2E testing framework
- **Page Object Model** - Organization pattern
- **Test Fixtures** - Centralized data

### Development Tools
- **pnpm** - Efficient package manager
- **ESLint** - Linting and code quality
- **PostCSS** - CSS processing

### Architecture Patterns
- **Component-Based** - Modular and reusable
- **Hook Pattern** - Shared logic
- **Provider Pattern** - Context and dependency injection
- **Repository Pattern** - Data abstraction
